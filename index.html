Weaver

A Unity plugin which allows you to avoid using magic strings to reference assets, animations, layers, navigation areas, scenes, and shaders. This makes your code more efficient and robust to improve development productivity as well as runtime performance.

[Pro] [Lite] [Documentation] [Forum] [mail@kybernetik.com.au]

UltEvents

A free Unity plugin which allows you to easily setup and configure persistent event callbacks via the Inspector. It serves the same purpose as the inbuilt UnityEvents, but has superior features with fewer restrictions and an improved Inspector interface.

[itch.io] [Unity Asset Store] [Documentation] [mail@kybernetik.com.au]
